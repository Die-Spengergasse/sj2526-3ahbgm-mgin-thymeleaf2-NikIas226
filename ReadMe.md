## How to Start the Project
1. Start the application.
2. Open a web browser.
3. Enter the following URL in the browser:
   http://localhost:8080/

## Medication List Features
1. **Add medication** – Adds a medication to the list.
2. **Edit medication** – Edits an existing medication in the list.
3. **Delete medication** – Deletes an existing medication from the list.
